# Fashion MNIST ANN vs CNN

This project demonstrates the implementation and comparison of **Artificial Neural Networks (ANN)** and **Convolutional Neural Networks (CNN)** on the **Fashion MNIST dataset**.  
The notebook includes model building, training, evaluation, hyperparameter tuning, activation function comparison, and confusion matrix analysis.

---

## 📌 Project Overview

The main objective of this project is to classify images from the **Fashion MNIST dataset** using deep learning models and compare their performance.

This project covers:

- Data loading and preprocessing
- ANN model implementation
- CNN model implementation
- Activation function comparison (**Sigmoid vs ReLU**)
- Learning rate tuning
- Model training and evaluation
- Prediction on test data
- Confusion Matrix visualization
- Performance comparison between ANN and CNN
